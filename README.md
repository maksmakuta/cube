# Cube

 Voxel rendering engine (like Minecraft).  
 Written in C++ and OpenGL.  
 In progress...

### Features

 - [ ] World
   - [ ] Generation
     - [x] Flat terrain
     - [ ] Noise generated terrain
   - [ ] Biomes support
   - [ ] Structures
     - [ ] Grass 
     - [ ] Trees
 - [ ] Rendering
   - [x] Minecraft-like camera system
   - [ ] Mesh creation based on chunk data
   - [ ] Real-time rendering
     - [ ] Shadows
     - [ ] Lighting
