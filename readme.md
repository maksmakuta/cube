# Cube

Voxel rendering engine 

### Features

- [ ] World generation
  - [ ] Chunk manager system
  - [ ] Dynamic loading
  - [ ] Multithreaded generation
  - [ ] Generator 
    - [ ] Multithread support
    - [ ] Biomes support
- [ ] 2D Rendering
  - [ ] UI Rendering
  - [ ] Layouts
    - [ ] Column
    - [ ] Row
    - [ ] Tabs
  - [ ] Widgets
    - [ ] Button
    - [ ] Label
    - [ ] Switch
    - [ ] Icon/Image
    - [ ] Input
- [ ] 3D Rendering
  - [ ] Sky
    - [ ] Sun/Moon cycle
    - [ ] Clouds
      - [ ] Flat
      - [ ] Detailed
  - [ ] Block Rendering
    - [ ] Filled
    - [ ] X-shaped
    - [ ] #-shaped
  - [ ] Support for chunk rendering
    - [ ] Multithreading mesh generator
- [x] Window
  - [x] Resizeable
  - [x] Load OpenGL functions
  - [x] Fill rendering

### Dependencies

 - OpenGL 4.6 Core profile
 - GLAD ([included](/libs/glad))
 - GLM
 - GLFW 3.x
 - STB ([included](/libs/stb))
   - stb_image
   - stb_image_write
   - stb_truetype
   - stb_perlin
